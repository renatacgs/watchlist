# watchlist 🎬🌸

Sistema de gerenciamento de filmes e séries desenvolvido em **PHP** com arquitetura **MVC**.

- Cadastro e Login de usuários.
- Armazenamento de senhas com hash **Bcrypt**.
- Dashboard personalizada por usuário.
- **CRUD Completo** de filmes/séries (Adicionar, Listar, Remover).
- **Linguagem:** PHP 8.2
- **Banco de Dados:** MySQL
- **Arquitetura:** MVC (Model-View-Controller)
- **Design Patterns:** Singleton (Conexão) e DAO (Data Access Object)
- **Estilização:** CSS3 (Tema Rosa Pastel)

## 📋 Como rodar o projeto
1. Clone o repositório na pasta `htdocs` do seu XAMPP.
2. Importe o arquivo `database.sql` no seu **phpMyAdmin**.
3. Certifique-se de que o Apache e MySQL estão ativos.
4. Acesse `http://localhost/nome-da-sua-pasta/index.php`.
